# utils

  • 调用方式：

    • 通过应用实例调用，具体如下：

      app.utils;

    • 通过 hCore 函数对象调用，具体如下：

      hCore.utils;

  • 用法：

    内置的工具函数。

# utils.clone

  • 调用方式：

    utils.clone(value);

  • 参数：

    • { String | Object | Number } value 拷贝的目标值

  • 返回值：

    • { String | Object | Number } value 拷贝之后的值

  • 用法：

    对目标值进行浅拷贝，即拷贝前后的两个值是等价的。

  • 示例：

    var objects = [{ a: 1 }, { b: 2 }];

var shallow = app.utils.clone(objects);
console.log(shallow[0] === objects[0]);
// => true

# utils.deepClone

  • 调用方式：

    utils.deepClone(value);

  • 参数：

    • { String | Object | Number } value 拷贝的目标值

  • 返回值：

    • { String | Object | Number } value 拷贝之后的值

  • 用法：

    与 clone 方法类似，只是会对目标值进行递归拷贝，即拷贝前后的两个值是不等价的。

  • 示例：

    var objects = [{ a: 1 }, { b: 2 }];

var deep = app.utils.deepClone(objects);
console.log(deep[0] === objects[0]);
// => false

# utils.extend

  • 调用方式：

    utils.extend(object, [sources]);

  • 参数：

    • { Object } object 要扩展的目标对象

    • { (...Object) } sources 扩展源

  • 返回值：

    • { Object } object 扩展之后的目标对象

  • 用法：

    对目标对象属性进行扩展，扩展源仅限于对象内的可枚举属性。扩展源支持多个，从左至右依次开始处理，如果之前的扩展源有相同属性会被覆盖。

    注意：extend 方法会修改要扩展的目标对象。

  • 示例：

    function Foo() {
  this.a = 1;
}

function Bar() {
  this.c = 3;
}

// 非可枚举属性
Foo.prototype.b = 2;
Bar.prototype.d = 4;

app.utils.extend({ a: 0 }, new Foo(), new Bar());
// => { 'a': 1, 'c': 3 }

# utils.isArray

  • 调用方式：

    utils.isArray(value);

  • 参数：

    • { * } value 要检测的目标值

  • 返回值：

    • { Boolean } 如果目标值是一个数组，则返回 true，否则返回 false

  • 示例：

    app.utils.isArray([1, 2, 3]);
// => true

app.utils.isArray(document.body.children);
// => false

app.utils.isArray("abc");
// => false

# utils.isFunction

  • 调用方式：

    utils.isFunction(value);

  • 参数：

    • { * } value 要检测的目标值

  • 返回值：

    • { Boolean } 如果目标值是一个函数，则返回 true，否则返回 false

  • 示例：

    app.utils.isFunction(() => 123);
// => true

app.utils.isFunction(/abc/);
// => false

# utils.isPlainObject

  • 调用方式：

    utils.isPlainObject(value);

  • 参数：

    • { * } value 要检测的目标值

  • 返回值：

    • { Boolean } 如果目标值是一个普通对象，则返回 true，否则返回 false

  • 用法：

    有两种场景，isPlainObject 会返回 true。一种是目标值是基于 Object 构造函数创建的对象，另一种是 [[Prototype]] 为 null 的对象。

  • 示例：

    function Foo() {
  this.a = 1;
}

app.utils.isPlainObject(new Foo());
// => false

app.utils.isPlainObject([1, 2, 3]);
// => false

app.utils.isPlainObject({ x: 0, y: 0 });
// => true

app.utils.isPlainObject(Object.create(null));
// => true

# utils.uniqueId

  • 调用方式：

    utils.uniqueId(prefix);

  • 参数：

    • { String } prefix id 前缀

  • 返回值：

    • { String } 唯一的 id 值，如果指定 prefix，则返回一个前缀为 prefix 的唯一 id

  • 示例：

    app.utils.uniqueId("contact_");
// => 'contact_104'

app.utils.uniqueId();
// => '105'
