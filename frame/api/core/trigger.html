# trigger

  • 调用方式：

    • 通过应用实例调用，具体如下：

      app.trigger(type, data);

    • 通过 hCore 函数对象调用，具体如下：

      hCore.trigger(type, data);

  • 参数：

    • { String } type 事件名称

    • { String | Object | Number } data 事件触发时传递的数据

  • 用法：

    触发自定义事件。

  • 示例：

    app.on("some-event", (data) => {
  // do something with data
  console.log("some-event fired");
});

// 触发 some-event，传递数据 data
app.trigger("some-event", {
  id: "123",
});
