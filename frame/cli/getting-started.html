# 快速上手

# 环境准备

命令行工具是基于 Node.js 开发的，要求提前安装好 Node.js 环境，并确保 Node.js 版本在 10.13.0 或以上。

执行以下命令可以查看当前的 Node.js 版本：

$ node -v
v10.15.3

# 脚手架安装

执行以下命令可以将脚手架安装到全局：

$ npm config set registry http://192.168.102.232:6060/repository/npm-public/
$ npm i @hui/cli -gd

执行以下命令可以检查安装是否成功：

$ hui

 hui <command> [option]

命令：
  hui create                     创建并初始化一个新的项目
  hui generate <path> [options]  生成一个新的视图组件  [aliases: g]
  hui dev [options]              启动开发环境的内置服务器
  hui build [options]            执行生产环境的编译集成任务
  hui serve <dir> [options]      使用内置服务器为指定目录提供静态资源服务
  hui service [options]          查看和管理当前环境的内置服务

选项：
  --help, -h     显示帮助信息  [布尔]
  --version, -v  显示版本号  [布尔]

# 业务开发

统一框架 2.x 屏蔽了底层的技术细节，极大的降低了开发者的入门门槛，按照以下的步骤即可快速上手进行业务开发：

  1. 项目创建
$ hui create
  1. 选择项目类型，指定项目名称，完成项目创建
? 请选择要创建的项目类型: app
? 请输入要创建的项目名称: demo
? 是否使用内置的路由引擎自动处理路由关系？ (Y/n) y
  1. 创建一个业务视图 business
$ cd demo
$ hui g business
  1. 开发调试
$ hui dev
  1. 构建集成
$ hui build
  1. 本地部署
$ hui serve ./dist/
