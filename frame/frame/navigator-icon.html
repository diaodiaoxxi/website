# 图标

框架导航 顶部栏、侧边栏 自定义图标目前按照 设计规范统一使用 iconfont 的 font-class 引用方式来配置；

首先需要挑选别人图库库的图标，或者使用 svg 图标生成 iconfont 字体集；

可使用 阿里 iconfont 或 iconmoon 统一管理 项目图标来生成 iconfont，推荐使用 阿里 iconfont；

框架导航的侧边栏目前只有第一级的菜单项支持自定义图标

# 项目要求

  1. 图标项目创建好后,修改 fontClass 避免与应用内其他项目图标冲突

  1. 项目名称修改为，框架导航内所属 系统名称-项目名称， 如 system-project-icon-，system-project-iconfont 框架导航相关图标为: h-layout-icon,项目名称避免使用 h-layout 以免与框架导航内图标冲突

# 3. 框架导航页面模板配置自定义图标

  • SystemInfo 下的 MenuInfo icon 属性定义为 iconfont.css 中的 FontClass 名称。
  • FontClass 名称和预览可以在下载的自定义图标字体集中的 demo_index.html 中查看。
  • 在框架导航中手动格式化 appSystemList ，格式化过程可以参考 框架导航对接操作员中心接口工程示例。
