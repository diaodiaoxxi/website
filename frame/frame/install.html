# 安装

在框架应用创建的过程中，hui-micro-app 会作为核心依赖自动进行安装。在应用创建完成后，开发者也可以执行以下命令安装指定版本或进行版本升级。

# 安装指定版本 x.x.x
$ npm i hui-micro-app@x.x.x -d

# 安装最新版本
$ npm i hui-micro-app@latest -d
