# UED统一设计规范之HUI Design Pro页面模板

# HUI Design

HUI Design是恒生设计体系，它服务于恒生B端企业级产品的设计体系，基于组件、导航框架、布局等模块化解决方案，让业务开箱即用，提升研发效率，让UX/UI保持良好的一致性。

# 页面模板

页面模板将秉承HUI Design的设计价值观，在设计规范和基础组件的基础上，继续向上构建，提炼出典型的页面模板以及配套的设计资源，进一步帮助“用户”提升在企业中后台产品中的体验。随着“用户”的不断反馈，我们将不断地迭代，持续收集、分析、抽取、总结出更多的使用共性需求和设计模式，持续输出设计资产。

# 布局

“混沌是自然的法则，而秩序是人类的梦想”。——亨利·亚当斯

在HUI Design Pro页面模板的设计中，空间布局是体系化视觉设计的出发点，在企业中后台产品中，界面的空间布局要形成有规律可循的体系化架构展开。基于这样的体系化规律，探索UI设计中的动态空间秩序之美，形成了HUI Design的界面布局方式，为设计师，开发者，产品经理等多角色用户构筑具备理性之美的空间布局创造条件。

图中可以看出，1920x1080分辨率是占比最高的，而我们最终采用1366x768WXGA的分辨率是考虑到我们的用户和设计者更多适配性，同时，1366x768的显示比例是16:9，在通常出现的13～15寸笔记本电脑、常规投影仪等各种载体上显示时，均为较佳效果。

# 3、单位尺寸

HUI Design通过网格体系来实现视觉体系的秩序之美。网格的基数为8px，区域间隔等推荐(8+4n)px,除了符合偶数的思路外，同时能够更好地匹配多数主流显示设备。

# 页面模板的分类与探索

在恒生众多中后台企业中，经过调研，我们将页面模板梳理出7大模块40余个页面，同时，我们仍在搜集更多的使用场景，抽取共性的部分作为统一的页面模板资源库，将对两大类产品有重大帮助。第一类，已有产品的在众多复杂页面中有规律可循，有页面可套，第二类，新产品的开发中，设计资源缺乏的情况下，一定程度上保证了输出页面的规整与理性之美。

# 是起点，是启发，非限制

HUI Design在空间布局的规划和页面模板的资源输出，更多的是在于引导设计者如何做到“规范”、“统一”和“更好”。建议以8+4n的数模为变化基础，通过排列组合的方式，形成无数种衍生页面设计方案。所有这些成果并非限制设计产出，其目的是为了在HUI Design的设计价值观、设计原则和设计模式指引下，实现更加优雅合理的界面布局，更具体系化的设计输出。
