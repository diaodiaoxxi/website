# PPT美化设计需求经验总结（下）

# 引言

在《PPT美化设计需求经验总结（上）》中介绍了如何对需求进行分析、评估工作量、确定优化方向，在本文中我们就要考虑如何运用一些经验和技巧去完成一套ppt的美化。

PPT的美化具体会涉及到哪些内容？

一套完整的PPT文件一般包含：片头、动画、PPT封面、前言、目录、过渡页、图表页、图片页、文字页、封底、片尾动画等，我们日常工作中比较常接触到的有PPT封面、目录、过渡页、图表页、图片页、文字页、封底；这些页面中采用的素材有：文字、图片、图表、动画、声音、影片等。

本文将从配色、文字处理、图片处理、内容优化、模板设计等角度对各中素材的处理手法做一个解析（出于对内容的保护，下面图片做了模糊处理）。

# PPT色卡选取方法

在已确定的模板中提取配色方案是一种非常安全、快速、高效的方法。

取色对象一般为模板的背景色、主视觉图形。选择模板中最大比重的一种颜色作为主色调，第二大比重的配色作为辅助色。主辅色一般为90°以内的类似色、60°以内的邻近色；在PPT的演讲中通常会有需要重点突出的内容，可以再选取1-2个对比色（120°到150°）或者互补色（180°），创造出强对比的色彩感受，可以增强感官上的刺激性。但是最好不要大面积使用。

这样就能快速得到一套基础色卡，可以覆盖大部分场景的使用，在颜色不足的时候，可以通过使用同位色或者通过改变基础色卡的纯度和明度，延伸出其他色彩搭配。

# 二、文字处理

PPT字体建议中文和英文分别统一用一种字体，英文推荐Arial，所有系统都有安装显示效果不错。中文推荐使用微软雅黑，windows电脑默认字体，大部分苹果电脑也会安装，显示效果一致，比较安全。另外还可以选用思源黑体，相比微软雅黑更优雅一些，开源免费，但是需要确认最终投屏和对PPT进行评审的成员都预安装这个字体。

或者可以在PPT中嵌入字体，就可以不受电脑系统中是否有这个字体的限制，但是文件会相对应变大。

# 三、图片的处理手法

图片在PPT中也是比较常见的元素，尤其在大型演讲中，用图片会更有叙事性引起观众共鸣。但我们拿到的PPT往往图片颜色、大小不一致，会造成页面杂乱无章、色彩混乱。比如下图三张配图，首先长宽比不一，其次，第一张图是冷色调，后两张是暖色调，在色彩上形成了不一致。

我们可以通过下面几种方法对图片进行优化：

# 1、通过着色功能调整图片色调

在实际操作中，我们可以通过着色功能把图片调整为同一种色调，可以选用窗体中的推荐色，也可自定义色调。自定义色调可以从PPT的主色卡中选择并且调整明度、对比度获得。

# 2、通过裁切功能调整图片大小，保持视觉平衡

在实际操作中，我们可以先把图片按原比例调整为同一高度/宽度，在这个基础上再对图片进行裁切，裁剪成相同的宽度/高度。

# 3、用统一的图形来增加设计感

方正的矩形会使画面更正统、稳定，如果有时候需要体现一点设计感，我们还可以运用裁切操作将图片调整为各种菱形、圆形、三角形等等。可以通过PPT本身的主题和气质灵活使用。

# 4、改变图文比例增加视觉冲击力

这种处理场景常常在大型演讲中使用，能营造出大气的感觉，比较适合应用在在一些表达关键观点的过渡页、结束页。

# 四、内容的二次加工，注重图形化表达

以上提到的处理都是针对需求方已确定的内容进行优化，而在时间比较充裕的情况下，设计师还需要从体验的角度去反推内容，帮助演讲者更好地组织信息。最常见的场景就是我们拿到的待优化的内容是一大段文字，我们在日常生活中都有这样的感知，一段很长的文字往往读下来是比较难的，但是拆分成多个短句，会更容易接收并理解。我们可以从两个角度去对长段文字进行切割：

1、这段文字短句之间有没有内在逻辑，是否可以通过视觉的方式把这种逻辑、层次表现出来，比如下方案例想表达一个进程，我们可以用时间轴的方式去做优化，视觉上更清晰，阅读更顺畅，理解更容易。

2、大脑处理视觉内容的速度比文字内容快60000倍，视觉内容可以说是一图胜千言。适当地用图说话，可以加快理解，甚至加深记忆（右脑将看到，听到和想到的事物，全部转化为图像进行思考和记忆），从视觉角度来讲，画面也更饱满。

# 五、使用母版进行设计

因为PPT的优化工作常常会有好几轮的修改，演讲者可能会在你优化的基础上做一些增、删、编辑的操作，而他们往往对于视觉改动的感知比较薄弱，如果不用模板做固定，可能需要你二次修改的PPT就会变得面目全非，二次修改的工作量就会加倍。

所以我们自己设计模板的时候，建议将重复出现的并且固定位置的元素在母版中设定好位置和样式，这样其他人在修改的时候就不会改动字号和位置，即使有移动，重置一下版式即可。如果设计上有改动，比如说快要开会了，但是到现场发现原来设计的标题字号不够大，直接在母版里修改，所有相关的页面就能一并批量调整，能很大程度地节约工作量。

进入视图-幻灯片母版界面即可对母版进行设计。常见的固定元素有背景、标题、页眉、页脚，图片直接放置在固定位置即可，文字类型通过占位符插入，并且设定好字号、颜色、字体等样式，再回到常规视图，给每页PPT选择相应的板式即可。

# 总结

想要高效地完成一个PPT美化需求，需要有“术”有“法”，处理图文信息、使用母版都是一些高效实用的技巧“术”，理解和消化内容，并且用恰当的视觉方式表现则为“法”，这些都需要在日常工作中积累更多的经验和更多灵活的思考。
