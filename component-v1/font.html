# Font 字体

# Font 概述

hui 的 CSS 对字体进行了统一规范，力求在不同平台、浏览器下能显示出其最佳的效果。我们推荐 macOS（iOS）优先的策略，在不支持苹方字体的情况，使用备用字体。

# 字体预览

  • 中文字体

  • 英文字体

# 字体代码

font-family: "Helvetica Neue",Helvetica,"PingFang SC","Hiragino Sans GB","Microsoft YaHei","微软雅黑",Arial,sans-serif;
