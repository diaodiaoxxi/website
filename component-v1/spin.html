# Spin 概述

当区块正在获取数据中时可使用，适当的等待动画可以提升用户体验。

# Spin 代码演示

# 基础用法

最简单使用 Spin 的方法。

# 各种尺寸

通过设置 size 属性为 large 和 small 将 Spin 设置为大和小尺寸，不设置为默认（中）尺寸。

# 居中固定

在容器内部垂直居中固定，需要父级有 relative 或 absolute。

# 自定义内容

自定义 Spin 的内容，可以是简单的文字，也可以是很复杂的动画。

# 状态切换

控制 Spin 组件的显示和消失。

# Spin props

属性 说明 类型 默认值
