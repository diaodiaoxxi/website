# Log props

属性 说明 类型 默认值
logs 每次传入面板内的日志 Array []
maxSize 面板内显示日志条数的最大阈值(超过最大阈值，将去掉日志前最大阈值的1/4条日志) Number 10000
height 日志显示面板高度，不包括footer（尾部） Number 400
isClear 是否显示 footer（尾部） Boolean true
