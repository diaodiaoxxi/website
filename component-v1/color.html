# Color 色彩

# Color 概述

hui 默认使用以下调色板的颜色作为设计和开发规范，以保证页面和组件之间的视觉一致。

# Color 示例

# 主色

hui 使用较为安全的蓝色作为主色调，其中 Light Primary 常用于 hover，Dark Primary 常用于 active。

Primary
#0275d8
Light Primary
#43a7fd
Dark Primary
#025aa5

# 辅助色

辅助色是具有代表性的颜色，常用于信息提示，比如成功、警告和失败。

#d9534f
