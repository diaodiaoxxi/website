# TreeGrid2.0 树表格

# TreeGrid2.0 概述

主要用于展示大量结构化数据。
支持排序、筛选、分页、自定义操作、导出 csv 等复杂功能。

注意：非 template/render 模式下，需使用 u-tree-table。

# 基础用法

表格的最简单用法。

# 斑马纹

设置属性 stripe ，表格会间隔显示不同颜色，用于区分不同行数据。

# 带边框,不显示表头

border 添加表格的边框线。
show-header 是否显示表头

# 特定样式

行：通过属性 row-class-name 可以给某一行指定一个样式名称。 列：通过给列 columns 设置字段 className 可以给某一列指定一个样式。 单元格：通过给数据 data 设置字段 cellClassName 可以给任意一个单元格指定样式。

# 固定列

通过给数据 columns 的项设置 fixed 为 left 或 right，可以左右固定需要的列。当横向内容过多时可以使用。

# 多选

通过给 columns 数据设置一项，指定 type: 'selection'，即可自动开启多选功能。 给 data 项设置特殊 key _checked: true 可以默认选中当前项。 给 data 项设置特殊 key _disabled: true 可以禁止选择当前项。
设置checkStrictly为 false 则不联动, 默认联动
正确使用好以下事件，可以达到需要的效果：

  • @on-select，选中某一项触发，返回值为 selection 和 row，分别为已选项和刚选择的项。
  • @on-select-all，点击全选时触发，返回值为 selection，已选项。
  • @on-selection-change，只要选中项发生变化时就会触发，返回值为 selection，已选项。

# 自定义列模板

通过给 columns 数据的项，设置一个函数 render，可以自定义渲染当前列，包括渲染自定义组件，它基于 Vue 的 Render 函数。
render 函数传入两个参数，第一个是 h，第二个是对象，包含 row、column 和 index，分别指当前单元格数据，当前列数据（columns 内列数据），当前是第几行。
📺 学习 Render 函数的内容
render 函数本质返回的是字符串，u-tree-table 组件在内部对其进行了编译，如果使用了自定义组件 ，需要特别注意上下文，编译后的自定义组件，默认的上下文是 Table 所在的上下文，如果想让组件在指定的实例下编译，可以给 Table 设置属性 context 来指定上下文，比如本例指定当前路由页为上下文。一般情况不需要此配置，但如果你把 Table 作为一个 slot 封装在其 它组件里，这时 context 属性就很有用，比如父级是 $parent，根组件 $root。

# 多表头配固定列

::: tip 注意：只有多行存在时表格才可以跨行 :::

# canDrag 与 canMove 属性

  • canDrag: 使用后可以移动列头之间的边线来改变列宽（需要在 border 显示竖线的情况下使用）
  • canMove:使用后可以拖拉列头改变列的前后顺序
    同时可以使用以下方法监听过程
  • on-drag:改变列宽时触发,参数为（新的宽度，列名）
  • on-move:改变列排序时触发，参数为（列初始 index，改变后新的位置 index）

# 汇总 ：isSum 和 summationData

isSum ：在 column 中声明 isSum，会在汇总行生成该列的汇总数据，如需要自定义汇总算法，则需要通过 sumationData 来显示
summationData ：在 tabale 中插入一条汇总数据，格式与 data 一致， 直接指定汇总数据
sumType:"money" ：在列中配置了 isSum 之后，可以使用 sumType:"money" 指定汇总结果格式为金额

# 加载中 ：loading

loading ：通过控制 loading 为 true 或 false 来显示 loading 页面

# 使用 checked 标识选中状态

data 中默认添加 checked 属性记录当前选中状态
expand 属性记录当前展开状态

# 悬停显示完整列头信息:headerTooltip

column 中设置 headerTooltip：true，可以实现鼠标悬停显示列头完整信息

# 使用 Ctrl 和 Shift 多选

在多选模式下，实现按住 ctrl 和 shift 多选, 键盘上下键实现单选切换, shift + 键盘上下键实现多选 需要开启 highlight-row 、checkStrictly 设置为 false

# render 函数实现自定义渲染 title

在某些特殊场景下，鼠标移上表格时显示的 title 需要显示成 render 函数处理之后的值，这种场景可以通过 render 函数实现。 用 render 函数时只能通过 render 函数去实现每列的 title
用 render 函数不需要再去配置 showTitle 属性和 column 的 ellipsis 属性

# 增删改

handleAddData(rowid, arr): 新增数据
handleDelData(rowid):删除数据
setEditableRow(row): 编辑数据
cancelEditableRow(row): 保存数据 方法参数： row: 行数据
rowid: 当前行 rowid, 若传值为"root"则会增加根结点数据，否则是给 rowid 的行增加子节点数据 arr: 新增数据 注意：仅在 editType 为'rowEdit'时生效

    # TreeGrid2.0 events

    事件名 说明 返回值
    on-current-change 开启 highlight-row 后有效，当前行选中的时候会触发
  • currentRow：当前高亮行的数据
  • oldCurrentRow：上一次高亮的数据
    • on-select-all 在多选模式下有效，点击全选时触发
  • selection：已选项数据
    • on-selection-change 在多选模式下有效，只要选中项发生变化时就会触发
  • selection：已选项数据
  • rowid：已选项数据 id 合集
    • on-row-click 单击某一行时触发 当前行的数据和 rowid
    on-row-dblclick 双击某一行时触发(会触发单次点击事件)
    on-expand 展开或收起某一行时触发
  • row：当前行的数据
  • status：当前的状态
    • on-scroll 表格内纵向滚动条滚动时触发 返回第一个参数为当前纵向滚动条距离底部的距离（为 0 时表示滚动条已经滚动到最底部）；第二个参数是当前纵向滚动条距离顶部的距离【返回顶部距离的参数】；第三个参数为滚动方向（"x"代表水平滚动，"y"代表垂直滚动）
    on-drag 拖动表头是触发 返回当前拖动表头的宽度，和返回当前拖动表头 key
    on-editinput-change input 组件选择项发生变化时触发 返回 3 个参数，分别为当前值，当前项在表格中的横坐标和纵坐标
    on-editinput-enter inpu 输入框输入 enter 键时触发 返回 3 个参数，分别为当前值，当前项在表格中的横坐标和纵坐标
    on-editinput-blur inpu 输入框失去焦点时触发 返回 3 个参数，分别为当前值，当前项在表格中的横坐标和纵坐标
    on-editarea-change area 文本框选择项发生变化时触发 返回 3 个参数，分别为当前值，当前项在表格中的横坐标和纵坐标
    on-editarea-blur area 文本框失去焦点时触发 返回 3 个参数，分别为当前值，当前项在表格中的横坐标和纵坐标
    on-money-change money 组件选择项发生变化时触发 返回 3 个参数，分别为当前值，当前项在表格中的横坐标和纵坐标
    on-money-enter money 组件框输入 enter 键时触发 返回 3 个参数，分别为当前值，当前项在表格中的横坐标和纵坐标
    on-money-blur money 组件失去焦点触发 返回 3 个参数，分别为当前值，当前项在表格中的横坐标和纵坐标
    on-editselect-change select 组件选择项发生变化时触发 返回 3 个参数，分别为当前值，当前项在表格中的横坐标和纵坐标
    on-editdate-change date 组件选择项发生变化时触发 返回 3 个参数，分别为当前值，当前项在表格中的横坐标和纵坐标

    # TreeGrid2.0 methods

    方法名 说明 参数
    getTreeSelection 返回已选中的项(适用于多选的情况，返回 check 为 true 的已加载节点数据) this.$refs.xxx.getTreeSelection();
    cancelEditableRow 取消当前行的编辑状态 this.$refs.xxx.cancelEditableRow('row');参数 row 为当前节点信息
    setEditableRow 设置当前行为可编辑状态 this.$refs.xxx.setEditableRow(row);参数 row 为当前节点信息
    handleAttrChange 动态改变属性（checked、expand、_isHighlight、_disabled） this.$refs.xxx.handleAttrChange(rowId, attr, status);rowId: 行唯一表示_rowid,attr: 要改变的属性,status: 改变属性状态
    handleAddData 新增数据(只在行编辑的情况下可用) this.$refs.xxx.handleAddData(rowid, arr); arr:新增数据，rowid: 如果是’root‘, 则是添加第一级，如果是行 rowid，则是为该行添加的子元素
    handleDelData 删除改行数据 this.$refs.xxx.handleDelData(rowid); rowid: 行 rowid

    # TreeGrid2.0 column

    列描述数据对象，是 columns 中的一项

    属性 说明 类型 默认值
    type 列类型，可选值为 index、text、textArea、money、card、select、date、time、selectTree String -
    title 列头显示文字 String #
    key 对应列内容的字段名 String -
    width 列宽 Number -
    align 对齐方式，可选值为 left 左对齐、right 右对齐和 center 居中对齐 String left
    className 列的样式名称 String -
    fixed 列是否固定在左侧或者右侧，可选值为 left 左侧和 right 右侧(右侧仅支持操作列固定) String -
    hiddenCol 是否隐藏当前列 Boolean false
    render 自定义渲染列，使用 Vue 的 Render 函数。传入两个参数，第一个是 h，第二个为对象，包含 row、column 和 index，分别指当前行数据，当前列数据（columns 内列数据），当前行索引。 Function -
    headerTooltip 鼠标指针悬停表头单元格时浮出提示信息展示完整内容 Boolean false
    isSum 列是否支持汇总，开启列汇总，表格自动计算该列累计结果显示在汇总行。如需使用自定义汇总算法请使用 summationData 方式开启汇总功能 Boolean false
    sumType 汇总结果格式化方式，默认数字类型，配置'money'，则按金额进行格式化并保留两位小数，如 1,000.00 String -
    minWidth table 自适应时，列的最小宽度 Number --
    type = text 属性字段参考 input 组件, 通过 typeParams 绑定 - -
    type = textArea 属性字段参考 textArea, 通过 typeParams 绑定 - -
    type = money 属性字段参考 typefield 组件, 通过 typeParams 绑定 - -
    type = card 属性字段参考 typefield 组件, 通过 typeParams 绑定 - -
    type = select 属性字段参考 select 组件, 通过 typeParams 绑定 - -
    type = date 属性字段参考 date 组件, 通过 typeParams 绑定 - -
    type = time 属性字段参考 time 组件, 通过 typeParams 绑定 - -
    type = selectTree 属性字段参考 selectTree 组件, 通过 typeParams 绑定 - -
    treeNode 节点收缩展开按钮将显示在第一个设置该属性为 true 的列上 Boolean false
    columnEdit 列是否可编辑 Boolean false
    typeParams 表单列上表单的参数配置 Object {}
    option select 下拉框选项 Array []
    hiddenOther 是否隐藏单元格内原有值 Boolean false

    # TreeGrid2.0 multiLevel

    列描述数据对象，是 multiLevel 中的一项

    属性 说明 类型 默认值
    title 多级显示文字 String #
    cols 当前表头所占跨列数 Number 1
    rows 当前表头所占跨行数，被跨行单元格右移动 Number 1
    align 对齐方式，可选值为 left 左对齐、right 右对齐和 center 居中对齐 String left
    className 表头单元格样式名 String -
    hiddenCol 是否隐藏当前表头列 Boolean false
    fixed 表头标题是否固定在左侧或者右侧，可选值为 left 左侧和 right 右侧（仅支持操作列固定） String -

    # TreeGrid2.0 row

    data 数据对象的一项

    属性 说明 类型 默认值
    checked 是否勾选(在父子树节点联动下,如果勾选且当前节点为父树节点的情况下，子节点也会全部勾选) String #
    expand 是否展开子节点 Boolean false
    children 子节点属性数组 Array []
    leaf 叶子节点属性，设置 leaf 后该节点为叶子节点(children,loading 无效) Boolean false
    autoLoad 设置该节点可以在初始化时自动加载数据 Boolean false
    loading 设置该节点表示异步加载子节点数据,(为 true 表示正在加载子节点数据，需设置 load-data) Boolean false
