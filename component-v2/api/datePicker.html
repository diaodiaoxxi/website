# DatePicker2.0 日期控件

# DatePicker2.0 概述

概要说明

选择或输入日期，支持年、月、日期等类型

# DatePicker2.0 代码示例

# 基础用法

概要说明

  1. 设置属性 type = date，显示选择单日

  2. 设置属性 type = daterange 显示选择范围

  3. 设置属性 type = month 显示选择年月

  4. 设置属性 type = year 显示选择年份

# 日期格式

概要说明

设置属性 format 可以改变日期的显示格式

  1. type 为 date、daterange，支持格式 yyyy-MM-dd、 yyyy/MM/dd、yyyyMMdd

  2. type 为 month，支持格式 yyyy-MM、 yyyy/MM、yyyyMM

  3. type 为 year，支持格式 yyyy

# 快捷方式

概要说明

设置属性 options 对象中的 shortcuts 可以设置快捷选项（仅 type 为 date、daterange 支持）， 设置快捷选项，每项内容：

  1. text | String：显示的文案

  2. value | Function：返回指定的日期，如需自己控制逻辑，可不设置，并使用 onClick 回调

  3. onClick | Function：点击时的回调，参数为当前日期选择器的 Vue 实例，当需要自定义复杂操作时，可以使用

# 不可选日期

概要说明

设置属性 options 对象中的 disabledDate 可以设置不可选择的日期，disabledDate 是函数，参数为当前的日期，需要返回 Boolean 是否禁用这天

# 日期选择面板出现的位置

概要说明

通过设置属性 placement 为 ： 可选值为top、top-start、top-end、bottom、bottom-start、bottom-end。

# 是否禁用/只读

概要说明

  1. disabled 为 true 时不能弹出选择面板，日期不可手动修改

  2. readonly 为 true 时不能弹出选择面板，日期不可手动修改

# 隐藏清除按钮

概要说明

clearable 默认为 true，显示清除按钮，在数据不为空时，鼠标移入输入框时显示，false 不显示

# 事件使用实例

概要说明

  1. on-change 日期发生变化时触发；

  2. on-open-change 弹出日历和关闭日历时触发；

  3. on-select-range 在 type=daterange 时有效

# DatePicker2.0 API

# DatePicker props

属性 说明 类型 默认值
type 显示类型，可选值为 date、 daterange、month、 year String date
value 只用于 v-model 传入日期，yyyy-MM-dd 和 yyyy/MM/dd/，yyyyMMdd 形式字符串传入日期格式与 format 保持一致 String -
format 展示的日期格式 String 1. date | daterange 默认 yyyy-MM-dd，可选 yyyy-MM-dd 、 yyyy/MM/dd 和 yyyyMMdd;
2. month 默认 yyyy-MM, 可选可选 yyyy-MM 、 yyyy/MM 和 yyyyMM;
3. year 默认 yyyy
placement 日期选择器出现的位置，可选值为 top top-start top-end bottom bottom-start bottom-end String bottom-start
placeholder 占位文本 String 空
options 选择器额外配置，比如不可选日期与快捷选项，具体项详见下表 Object -
disabled 是否禁用选择器 Boolean false
clearable 是否显示清除按钮 Boolean true
readonly 完全只读，开启后不会弹出选择器 Boolean false
transfer 是否将弹层放置于 body 内，在 Tabs、带有 fixed 的 Table 列内使用时，建议添加此属性，它将不受父级样式影响，从而达到更好的效果 Boolean false
tabindex 设置输入框 tabindex String | Number -1

1. text | String：显示的文案
2. value | Function：返回指定的日期，如需自己控制逻辑，可不设置，并使用 onClick 回调
3. onClick | Function：点击时的回调，参数为当前日期选择器的 Vue 实例，当需要自定义复杂操作时，可以使用		 Array -
disabledDate 设置不可选择的日期，参数为当前的日期，需要返回 Boolean 是否禁用这天 Function -

# DatePicker events

事件名 说明 返回值
on-change 弹出框中选择发生变化/设置双向绑定值/手动输入日期时触发 已经格式化后的日期，比如 2016-01-01
on-open-change 弹出日历和关闭日历时触发 true | false
on-focus 获得焦点时触发 -
on-blur 获得焦点的状态下点击 DatePicker 之外的区域时触发 -
on-select-range 在返回选择时，选择日期触发，返回当前点击日期和状态,只在范围选择时有效 第一个参数表示当前日期
第二个参数表示返回状态，范围选择是第一次点击返回 false，第二次点击返回 true

# DatePicker methods

方法 说明
focus 通过 this.$refs.xxx.focus() 调用,使当前输入框获取焦点,并打开选择面板
