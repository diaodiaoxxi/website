# 内容规范

文档规范遵循 “约定优于配置” 的原则。我们都认同优雅的文档格式对浏览者的体验来说非常重要。所以接下来你所输出的所有文章都要严格遵守如下规范，每一条都要遵守，保证产出内容行文格式完全一致，才能方便的把 Markdown 内容拷贝出去到各大平台样式不会乱掉，不会出现奇怪字符。

# 配置

  • 导航或左侧菜单操作配置请在目录 .vuepress/siderbar.config/ 的文件中更新与调整
  • 其他配置在 .vuepress/config.js 文件中
  • 更多关于 vuepress 配置

# 命名

# 文件名/文件目录名命名

此处文件名包括 markdown、image、js 和 css 文件。

  • 英文，比如 manual.png
  • 小写，比如 manual.js
  • 多单词以半角转态短横杠来连接，比如 materiel-data-stream.md

其中，文件和文件夹的命令会应用到路由。比如文件路径 src/pagecook/manual/materiel-data-stream.md，那么其访问路由就是 /pagecook/manual/materiel-data-stream.html。

# 模板/组件命名

即存放在 .vuepress/tempaltes/ 和 .vuepress/components/ 目录下的文件，采用大驼峰方式，比如 NavigationLink.vue。

# 目录（路径）结构

  • 一级分类（目录）文件放在根目录下
  • 组件统一放到目录 .vuepress/components/ 下，比如 .vuepress/components/NavigationLink.vue
  • 模板统一放到目录 .vuepress/templates/ 下 .vuepress/templates/NavigationLink.vue
  • 单篇文章图片和该文章放在同级目录，比如文章 guide/article.md，那这篇文章用到的图片 demo.png 存放在 guide/article/demo.png，方便后期文章移动分类

# 中英文前后留出空格

  • ❌ 错误：12篇优质文章计划2020年完成，Javascript有关的话题有Node.js。
  • ❌ 错误：12篇优质文章计划2020年完成，Javascript有关的话题有 Node.js。
  • ❌ 错误：12篇优质文章计划2020年完成，Javascript 有关的话题有 Node.js。
  • ❌ 错误：12篇优质文章计划2020 年完成，Javascript 有关的话题有 Node.js。
  • ❌ 错误：12篇优质文章计划 2020 年完成，Javascript 有关的话题有 Node.js。
  • ❌ 错误：12 篇优质文章计划 2020 年完成，Javascript 有关的话题有 Node.js 。
  • ❌ 错误： 12 篇优质文章计划 2020 年完成，Javascript 有关的话题有 Node.js。
  • ✅ 正确：12 篇优质文章计划 2020 年完成，Javascript 有关的话题有 Node.js。

当数字、英文是句子开头，它前面不需留空格，当数字、英文是句子末尾，它后面不需留空格。

# 英文拼写要正确

英文的拼写，如 GraphQL 而不是 graphql，Node.js 而不是 nodejs，iPhone 而不是 iphone，Apple 而不是 apple，Java 而不是 java，Aliyun 而不是 aliyun，UED 而不是 ued，RDS MySQL 而不是 rds mysql，OSS 而不是 oss 等等，使用前，先网上查下正确使用。

# 不留空行

  • 标题内容之间不需要留空行
  • 文字描述与列表（无序列表或有序列表）之间，不需要留空行。

# 留空行

  • 段落与段落前后留空行

# 加粗不要加粗标点符号

❌ 错误：**HUI 是简单、灵活、快速、高效、轻量级的纯前端组件库。**
✅ 正确：**HUI 是简单**、**灵活**、**快速**、**高效**、**轻量级的纯前端组件库**。

# 文章的标题分级

通常情况下一个文档有一个一级标题，其余标题可用二级、三级和四级标题。

  • 标题和副标题用 ##、### 和 #### 表示，也就是 H2、H3 和 H4 标题
  • 手动输入 #### 再空格
  • 在 #### 下的列表，可用有序列表和无序列表

# 不要通过样式修改字体大小与颜色

❌ 错误：HUI 是简单、灵活、快速、高效、轻量级的纯前端组件库。

✅ 正确：HUI 是简单、灵活、快速、高效、轻量级的纯前端组件库。

特殊情况除外

# 段首不进行缩进

HUI 是简单、灵活、快速、高效、轻量级的纯前端组件库。

✅ 正确：

HUI 是简单、灵活、快速、高效、轻量级的纯前端组件库。
