# 转换规则（==）

  • 其中一个是Boolean类型 Boolean类型 => Number类型

  • String、Number类型 String类型 => Number类型

  • 对象、非对象 Object类型 => 通过Symbol.toPrimitive()、valueOf()、toString()转换

  • undefined == null

  • undefined、null不用转化成其他任何值

  • Number(null)值为0

  • Number(undefined)值为NaN

  • NaN == NaN

# [] == ![] true


[] == false  // 转换：!优先级高于==
[] == 0      // 转换：false转化为Number类型为0
'' == 0      // 转换：[]没有valueOf()，toString()为''
0 == 0       // 转换：''转化为Number类型为0

# [] == true false


[] == 1
'' == 1
0 == 1  // 得出为false

# !![] == true true


![]   // false  
!![] // true
true == true

# {} == !{} false

{} == false
NaN == false  //  {}.valueOf()为NaN
NaN == 0  // false

# [] == 0、![] == 0 true

0 == 0 // true
false == 0、0 == 0 // true

# + 加法运算符

转成string类型

  1. 字符串拼接：转成 String()
  2. 算术运算符：转成 Number()
  3. 复杂数据：
    • 先通过 valueOf() 转换成 原始数据，如果原始数据 不是 Number类型，则使用 toString()，将其转换成 String类型
    • 再将 String 转换成 Number类型

# object valueOf && toString


var a = {
  valueOf: function(){
    console.log('trigger valueOf')
    return 1
  },
  toString:function(){
    console.log('trigger toString')
    return 2
  }
}

2+a // 3   trigger valueOf
'2'+a // 21  trigger valueOf
'2' + [a] // "22" trigger toString
