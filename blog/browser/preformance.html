# Performance

# 概览页面

  1. FPS 图表上出现了红色块，那么就表示红色块附近渲染出一帧所需时间过久，帧的渲染时间过久，就有可能导致页面卡顿。
  2. CPU 图形占用面积太大，表示 CPU 使用率就越高，那么就有可能因为某个 JavaScript 占用太多的主线程时间，从而影响其他任务的执行
  3. 如果 V8 的内存使用量一直在增加，就有可能是某种原因导致了内存泄漏。

# 性能页面

  • Main 指标：记录 渲染主线程 的任务执行过程

  • Compositor 指标：记录 合成线程 的任务执行过程

  • GPU 指标：记录 GPU进程的主线程 的任务执行过程

  • Raster 指标：记录 光栅化线程池 的任务执行过程

  • Network 指标

  • Timings 指标

  • Frames 指标

  • Interactions 指标
